# Top-Down Betting Dashboard

Full Streamlit dashboard for top-down betting analysis.

## Run Locally
1. Clone repo
2. Create venv and activate
3. pip install -r requirements.txt
4. Set THEODDS_API_KEY
5. streamlit run app.py

## Deploy to Streamlit Cloud
1. Push repo to GitHub
2. Go to Streamlit Cloud → New app → select repo
3. Set THEODDS_API_KEY env var
4. Deploy → Live GUI
