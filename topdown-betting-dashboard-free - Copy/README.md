\# Interactive Top-Down Betting Dashboard



Streamlit dashboard for top-down betting analysis with full interactivity.



\## Features

\- Dynamic sport selection dropdown

\- Tabs for market types (moneyline, spreads, totals)

\- Filters: minimum EV, edge, ROI %

\- Column selection and sorting

\- Interactive fractional Kelly and bankroll

\- Debug info and number of events fetched



\## Run Locally

1\. Clone repo

2\. Create virtual environment

3\. pip install -r requirements.txt

4\. Set THEODDS\_API\_KEY

5\. streamlit run app.py
